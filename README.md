# Elastic-SIEM-Lab

## Introduction

Hello, and welcome to my Elastic SIEM project!

As part of my journey into cybersecurity and penetration testing, I wanted to gain hands-on experience with the defensive side of security. Understanding how attacks are detected, investigated, and mitigated is just as important as learning how they are performed. To help develop these skills, I decided to build an Elastic Stack Security Information and Event Management (SIEM) solution within my Proxmox home lab.

I created this repository to document my progress from start to finish, including the planning, deployment, configuration, and continuous expansion to my SIEM environment. Throughout this project, I will be collecting and analyzing logs from multiple systems, creating dashboards, developing detection rules, and simulating attacks to better understand how malicious activity is detected, investigated, and responded to.

My goal is not only to become familiar with Elastic SIEM, but also to strengthen my understanding of log analysis, incident investigation, and detection engineering. By documenting each step of the process, I hope this repository serves as both a learning resource for myself and a demonstration of my technical skills and growth throughout my cybersecurity journey.

## Project Goals

The primary objective of this project is to design, deploy, and maintain an enterprise-inspired SIEM environment using the Elastic Stack within my Proxmox homelab. Throughout this project, I aim to strengthen my skills in system administration, security monitoring, threat detection, and incident response while documenting each stage of the deployment.
